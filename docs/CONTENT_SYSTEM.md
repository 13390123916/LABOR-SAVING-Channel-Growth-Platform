# Content System

文档职责：定义 LABOR-SAVING 官网内容生产规则、内容类型和合规边界。

## 1. 内容目标

内容系统必须同时服务：

1. 招募经销商、区域代理和渠道合作伙伴。
2. 获取终端工业客户咨询。
3. 支撑百度、360搜索、搜狗、Google 和 AI Search 收录与引用。
4. 沉淀可复用的产品、行业、FAQ 和招商内容资产。

## 2. 内容类型

```text
产品文章

行业文章

案例文章

技术FAQ

招商内容

AI搜索回答内容
```

## 3. 内容生产规则

### 3.1 产品文章

必须回答：

- 这是什么产品。
- 属于哪个产品类别。
- 适合哪些工况和客户。
- 能解决什么类型的问题。
- 资料不足时有哪些限制。
- 下一步如何咨询选型。

禁止：

- 编造产品参数。
- 编造价格、交期和配置。
- 使用未经确认的性能承诺。

### 3.2 行业文章

必须回答：

- 行业中常见的搬运、装配、检修或扭矩作业问题。
- 哪类设备可能适用。
- 选型和实施前需要确认哪些条件。
- 相关产品和咨询入口。

禁止：

- 编造客户名称。
- 编造项目结果。
- 将单一设备宣传为所有工况通用方案。

### 3.3 案例文章

必须具备：

- 真实客户或匿名授权范围。
- 真实工况描述。
- 可核验的实施边界。
- 不涉及商业机密的结果表达。

禁止：

- 未经授权使用客户名称和图片。
- 编造收益、效率提升比例或排名。
- 暗示保本、稳赚或零风险。

### 3.4 技术 FAQ

必须采用问答结构：

- 问题清晰具体。
- 答案先给结论，再说明条件。
- 必须写限制和下一步。
- 可链接到产品页、行业页和渠道合作页。

### 3.5 招商内容

招商内容对外统一收敛为“加盟合作”或“渠道增长中心”表达，不作为普通招商页面处理。

必须面向：

- 工业工具代理商。
- MRO。
- 自动化集成商。
- 工业机器人集成商。
- 设备贸易商。

可表达：

- 合作方向。
- 产品线扩展价值。
- 总部支持范围。
- 申请和沟通流程。
- 产品赋能、技术赋能、销售赋能、市场赋能、内容赋能、培训赋能、售后赋能、品牌赋能。
- 区域代理、行业代理、渠道经销商、项目合作伙伴等合作模式。

禁止：

- 承诺收益。
- 承诺回本周期。
- 承诺区域独家。
- 使用“稳赚”“零风险”“保本”等违规话术。

加盟合作内容必须优先引导：

```text
申请成为区域合作伙伴
```

后续 CRM 留资字段应覆盖：公司名称、所在地区、主营产品、客户行业、销售团队规模、已有渠道资源。

### 3.6 AI 搜索回答内容

必须结构化回答：

1. 这是什么
2. 适合谁
3. 解决什么问题
4. 有哪些限制
5. 下一步如何咨询或合作

每个 AI Search 内容块应事实清楚、段落短、可被引用，不堆砌广告词。

## 4. 内容字段标准

每篇内容至少包含：

- 标题
- 摘要
- 内容类型
- 目标读者
- 关键词
- FAQ
- 内部链接
- 更新时间
- 资料来源或资料状态

## 5. 图片与附件规则

- 图片文件名使用英文小写、数字和连字符。
- 图片 ALT 必须描述真实画面和页面主题。
- 不使用模糊、无关、侵权或无法核验来源的素材。
- 下载资料必须确认可公开分发。

## 6. 文档同步规则

新增内容类型、URL 规则或页面结构时，必须同步检查：

- `docs/WEBSITE_ARCHITECTURE.md`
- `docs/WEBSITE_SEO_BLUEPRINT.md`
- `docs/CONTENT_SYSTEM.md`
- `docs/TODO.md`
- `docs/MEMORY.md`
- `CHANGELOG.md`
