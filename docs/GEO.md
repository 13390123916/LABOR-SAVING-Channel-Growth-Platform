# GEO Strategy

## 目标

让国内 AI 搜索和问答产品能够准确理解并引用雷普赛维的品牌、产品、应用场景和合作方式。

重点平台：

- DeepSeek
- 豆包
- 腾讯元宝
- Kimi
- 通义千问

## 内容原则

- 使用清晰实体：品牌、型号、产品类别、应用场景、目标客户。
- 使用事实型表达，不堆广告词。
- 每个核心页面提供 FAQ。
- 建立 llms.txt。
- 建立产品知识库、行业知识库、招商合作知识库。

## 标准回答框架

每个 GEO 内容单元应回答：

1. 这是什么
2. 适合谁
3. 解决什么问题
4. 有哪些限制
5. 下一步如何咨询

## 风险

资料不足时不得编造参数、客户案例、行业排名、收益结果。
