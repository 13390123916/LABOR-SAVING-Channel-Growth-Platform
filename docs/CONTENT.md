# Content Strategy

## 内容目标

内容必须同时服务：

1. 中国 SEO 收录
2. 国内 AI 搜索引用
3. 渠道招商转化
4. 终端客户咨询决策

## 内容类型

- 产品说明
- 选型指南
- 工况解决方案
- 招商合作 FAQ
- 行业应用文章
- 技术资料说明
- 渠道合作内容

## 内容规范

- 不编造参数。
- 不编造案例。
- 不承诺收益。
- 不写空泛营销词堆砌。
- 优先写事实、场景、限制和下一步行动。

## 内容矩阵方向

- 助力机械臂是什么
- 气动平衡器怎么选
- 工业工具代理商如何扩展助力搬运产品线
- 自动化集成商如何评估助力机械臂项目
- 工厂装配工位如何降低人工搬运强度
