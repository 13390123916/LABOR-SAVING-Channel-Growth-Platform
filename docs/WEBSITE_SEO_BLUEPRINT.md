# Website SEO/GEO Blueprint

文档职责：定义 LABOR-SAVING 官网 URL 规则、页面 SEO 字段标准和 GEO 优化规则。

## 1. URL 规则

### 1.1 基础规则

- URL 使用英文小写。
- 单词之间使用连字符 `-`。
- 分类页以 `/` 结尾。
- 型号页包含产品型号和产品类别。
- 行业页使用行业英文词组。
- 加盟合作主页面使用 `/partner/`。
- `/distributor/` 与 `/join/` 可作为投放短链或重定向路径，不作为主 SEO 页面。
- 知识中心统一放在 `/knowledge/` 下。

## 2. 页面 SEO 字段标准

每个可索引页面必须定义：

- Title
- Meta Description
- H1
- H2 结构
- 关键词
- 图片 ALT
- Schema
- 内部链接

## 3. 页面类型模板

### 3.1 产品分类页

- Title：产品类别 + LABOR-SAVING + 核心应用场景。
- Meta Description：说明产品类别、适用客户、典型工况和咨询入口。
- H1：产品类别名称。
- H2 结构：产品定义、适合谁、解决什么问题、型号入口、应用行业、咨询方式。
- 关键词：产品类别词、工况词、渠道合作词。
- 图片 ALT：品牌 + 产品类别 + 工况，不堆砌关键词。
- Schema：Product、BreadcrumbList、FAQPage。
- 内部链接：型号页、行业应用页、渠道合作页、FAQ。

### 3.2 产品型号页

- Title：型号 + 产品类别 + LABOR-SAVING。
- Meta Description：说明型号归属、适用方向和咨询方式，不编造参数。
- H1：型号 + 产品类别。
- H2 结构：是什么、适合谁、解决什么问题、限制与资料状态、相关产品、咨询方式。
- 关键词：型号词、产品类别词、应用场景词。
- 图片 ALT：型号 + 产品类别 + 真实画面描述。
- Schema：Product、BreadcrumbList、FAQPage。
- 内部链接：产品分类页、行业应用页、渠道合作页。

### 3.3 行业应用页

- Title：行业 + 工业助力与扭矩工具方案 + LABOR-SAVING。
- Meta Description：说明行业工况、设备需求和咨询入口。
- H1：行业应用名称。
- H2 结构：行业场景、典型问题、适用设备、实施限制、FAQ、咨询方式。
- 关键词：行业词、工况词、产品类别词。
- 图片 ALT：行业 + 工况 + 设备，不使用虚假客户名称。
- Schema：Article、BreadcrumbList、FAQPage。
- 内部链接：相关产品页、技术文章、渠道合作。

### 3.4 加盟合作页

- Title：LABOR-SAVING 渠道增长中心 + 工业装备合作伙伴招募。
- Meta Description：说明授权合作伙伴、工业智能搬运增长市场、合作模式、渠道赋能和申请入口。
- H1：成为 LABOR-SAVING 授权合作伙伴。
- H2 结构：首屏价值、市场机会、合作模式、八大渠道赋能、申请成为区域合作伙伴。
- 关键词：经销商加盟、区域代理、行业代理、工业工具代理、渠道经销商、工业装备渠道合作。
- 图片 ALT：品牌、合作场景或真实资质，不使用收益暗示。
- Schema：Organization、BreadcrumbList、FAQPage。
- 内部链接：产品中心、行业应用、联系方式。

### 3.5 解决方案页

- Title：解决方案主题 + 工业智能搬运或扭矩工具集成 + LABOR-SAVING。
- Meta Description：说明适用行业、典型工况、相关设备和方案咨询入口。
- H1：解决方案主题。
- H2 结构：适用场景、典型问题、设备组合、实施条件、相关行业、咨询方式。
- 关键词：解决方案词、行业词、工况词、产品类别词。
- 图片 ALT：行业 + 工况 + 设备组合，不使用虚假客户名称。
- Schema：Article、BreadcrumbList、FAQPage。
- 内部链接：产品页、行业应用页、加盟合作页、FAQ。

### 3.6 知识内容页

- Title：问题或主题 + LABOR-SAVING。
- Meta Description：概括答案、适用对象和下一步阅读方向。
- H1：文章主题或 FAQ 问题。
- H2 结构：定义、适合谁、解决什么问题、限制、下一步。
- 关键词：主题词、行业词、产品类别词。
- 图片 ALT：真实主题说明。
- Schema：Article 或 FAQPage。
- 内部链接：相关产品页、行业页、解决方案页、加盟合作页。

## 4. 加盟合作 SEO/GEO 专项

### 4.1 页面定位

加盟合作页面定位为：

```text
LABOR-SAVING 渠道增长中心
```

主 URL：

```text
/partner/
```

### 4.2 页面权重结构

| 模块 | 权重 | SEO/GEO 目标 |
| --- | --- | --- |
| 首屏价值 | 20% | 让代理商快速理解为什么加入 |
| 市场机会 | 20% | 承接工业自动化、重载搬运、风电、石化、智能制造搜索需求 |
| 合作模式 | 25% | 覆盖区域代理、行业代理、渠道经销商、项目合作伙伴 |
| 八大渠道赋能 | 20% | 建立区别于普通厂家招商的内容资产 |
| 留资转化 | 15% | 进入 CRM 线索采集 |

- 智能制造趋势

- 行业代理
- 渠道经销商

## 5. GEO 优化

针对平台：

- 百度
- 360搜索
- 搜狗
- Google
- AI Search

优化重点：

- 企业实体信息：品牌名称、英文名称、业务定位、目标客户、联系方式。
- 产品实体描述：产品类别、型号、适用场景、限制说明、咨询路径。
- 行业知识内容：行业场景、典型问题、设备适配方向和实施限制。
- FAQ 结构：每个核心页面提供可被 AI Search 摘取的问答单元。
- JSON-LD Schema：优先使用 Organization、Product、Article、FAQPage、BreadcrumbList。

## 6. GEO 回答结构

每个核心内容单元必须优先回答：

1. 这是什么
2. 适合谁
3. 解决什么问题
4. 有哪些限制
5. 下一步如何咨询或合作

## 7. 合规边界

- 不编造产品参数。
- 不编造客户案例。
- 不承诺收益、回本、保本或零风险。
- 不编造授权政策、区域独家、区域保护细则、价格体系或市场排名。
- 不用纯图片承载核心内容。
