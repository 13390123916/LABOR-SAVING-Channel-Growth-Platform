# Project PRD

项目名称：LABOR-SAVING Channel Growth Platform（雷普赛维渠道增长平台）

## 1. 项目定位

本项目是工业装备渠道增长平台，不是普通企业官网。

官网只是整个系统的获客入口，长期系统将覆盖 Website、SEO、GEO、CRM、Partner、Marketing、Knowledge Base 和 Dashboard。

## 2. 商业目标

### Priority 1：加盟合作

面向 B 端经销商、代理商、MRO、自动化集成商、工业机器人集成商和设备贸易商，获取渠道合作线索。

### Priority 2：终端客户询盘

面向终端制造企业，获取产品咨询、选型咨询、工况咨询和报价线索。

### Priority 3：品牌建设

通过中国 SEO、国内 AI 搜索优化和内容营销建立品牌搜索资产。

## 3. 目标客户

重点客户：

- 液压扭矩扳手代理商
- 液压拉伸器代理商
- 工业工具代理商
- 自动化集成商
- 工业机器人集成商
- MRO
- 设备贸易商

其次客户：

- 终端制造企业
- 工厂设备负责人
- 检修维护负责人
- 采购负责人

## 5. SEO 原则

仅按中国搜索环境设计，重点考虑：

- 百度
- 360
- 搜狗
- 神马
- 微信搜一搜
- 知乎
- 抖音
- 小红书

不以 Google SEO 作为默认设计依据。

## 6. GEO 原则

重点面向：

- DeepSeek
- 豆包
- 腾讯元宝
- Kimi
- 通义千问

内容必须结构化、事实型、可引用、可总结。

## 7. 技术栈

- Next.js
- TypeScript
- TailwindCSS
- Node.js
- MySQL
- Docker
- GitHub Actions

## 8. M0 范围

M0 只负责 Repository Bootstrap，不开发业务页面、不生成产品内容、不构建官网首页。

交付包括：

- 标准目录结构
- docs 文档体系
- GitHub 协作体系
- AI 协作体系
