# Roadmap

项目周期预计 12 到 24 个月，采用“平台底座先行、官网获客验证、CRM 渐进增强、内容资产规模化”的路线。

## M0 Repository Bootstrap

目标：建立企业级仓库基础设施。

范围：

- 目录结构
- GitHub 协作文件
- AI 协作文件
- 项目文档体系
- CI 基础检查
- 开源/私有授权声明

状态：已完成。

## M1 Website Foundation

目标：建立官网技术底座，不直接堆业务页面。

范围：

- Next.js App Router
- TypeScript
- TailwindCSS
- 基础布局系统
- SEO metadata 架构
- 静态资源规范
- 部署准备

## M2 Lead Capture Foundation

目标：建立线索采集与 CRM 前置数据结构。

范围：

- 询盘表单
- 加盟合作表单
- 线索来源追踪
- MySQL 数据模型
- API 校验
- 后续 CRM 分配字段

## M3 China SEO Foundation

目标：建立中国搜索引擎友好的内容与技术基础。

范围：

- 百度、360、搜狗、神马、微信搜一搜适配
- sitemap
- robots
- 面包屑
- 内链规范
- 关键词库
- 内容结构规范

## M4 GEO Foundation

目标：建立面向国内 AI 搜索的知识资产。

范围：

- llms.txt
- 产品实体库
- FAQ 标准答案
- 行业应用知识库
- AI 可引用内容规范

## M5 CRM Foundation

目标：将线索从表单收集升级为可管理、可分配、可跟进的 CRM。

范围：

- 线索后台
- 线索状态
- 区域识别
- 代理商分配
- 跟进记录
- 飞书 / 企业微信通知

## M6 Partner & Dashboard

目标：支持代理商协作和总部运营分析。

范围：

- 代理商账号
- 区域权限
- 线索协作
- 成交反馈
- 运营 Dashboard
- 内容与投放数据复盘
